# Papollo-Hospital-Analytics-Dashboard

## 🎯 Project Objectives
#### Analyze patient admission & discharge trends
#### Track bed occupancy (Private, General, ICU)
#### Evaluate hospital revenue & billing performance
#### Understand disease-wise patient distribution
#### Monitor doctor-wise feedback performance
#### Identify test usage trends (MRI, CT Scan, etc.)

## 🔄 Project Workflow

### 1️⃣ Data Collection

#### Healthcare dataset containing:
#### Patient_ID
#### Diagnosis
#### Doctor_Name
#### Admit_&_Discharge Dates
#### Billing_Amount
#### Test_Details
#### Feedback_Scores

### 2️⃣ Data Cleaning (SQL)

#### Removed duplicates
#### Handled missing values
#### Standardized columns (Diagnosis, Bed Type, Dates)
#### Formatted numeric fields

### 3️⃣ Data Analysis (SQL Queries)

#### Basic Queries (SELECT, WHERE, ORDER BY)
#### Aggregations (SUM, AVG, COUNT)
#### Grouping (GROUP BY, HAVING)
#### Filtering insights like:
#### Patients with specific diseases (Typhoid, Viral Infection)
#### ICU occupancy analysis
#### Revenue calculation
#### Doctor performance tracking

#### 🦠 Viral infections are most common diagnosis
#### 💰 CT Scan & MRI generate highest revenue
